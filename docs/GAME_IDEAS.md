# 💡 Game Ideas

Looking for inspiration? This file lists game ideas organized by difficulty and category. Pick one and submit it as a PR!

## How to Use This File

1. **Browse the ideas below** and find one that matches your skill level
2. **Check if it's already been submitted** by looking in the `games/` folder
3. **Follow the [Contributing Guide](../CONTRIBUTING.md)** to submit your game
4. **Not on this list?** You can submit any game you want — these are just suggestions

> Ideas marked with ⭐ **Great First PR** are especially good for first-time contributors.

## 🏷️ Great First PR Ideas

If you want the simplest possible first contribution:

1. **Dice Roller** (CLI/Python) — Just `random.randint()` and a loop
2. **Magic 8-Ball** (CLI/Python) — A list of answers and `random.choice()`
3. **Coin Flip** (CLI/Python) — 15 lines of code total
4. **Click Counter** (Web) — A button, a counter, a timer
5. **Tip Calculator** (Web) — Input field + basic math + display

Each of these can be completed in under 30 minutes and still counts as a real open-source contribution.
