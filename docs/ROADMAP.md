# 🗺️ Platform Roadmap

This document tracks what has been built and what's planned next.

---

## v1.0 — Foundation ✅ (Shipped)

- [x] Repository structure (games, translations, data)
- [x] Contribution guidelines and templates
- [x] Issue templates (game proposal, translation, bug, feature request)
- [x] PR templates (game submission, translation submission)
- [x] GitHub Actions automation (4 consolidated workflows: CI, Deploy, Community, Leaderboard)
- [x] Gamification system (XP, badges, levels, weekly leaderboard)
- [x] Security scanning and anti-spam protection
- [x] Welcome bot and contributor guidance
- [x] Example game submissions (CLI + Web)
- [x] Beginner's guide and documentation

## v2.0 — Web Platform ✅ (Shipped)

- [x] Astro static site deployed to GitHub Pages
- [x] Game catalog page with category/difficulty display
- [x] Individual game detail pages
- [x] Leaderboard visualization
- [x] Contributor profile pages with XP, level, and badges
- [x] Dark theme with responsive design
- [x] Auto-rebuild and deploy on push to main
- [x] SVG badge generator for contributor profiles

---

## Phase 3: Community Growth (Q2 2026)

### 🎯 Goals
- Reach 10+ community-submitted games
- Get first community translations
- Attract 10+ contributors

### 📋 Tasks
- [ ] Add monthly challenge system (themed game jams)
- [ ] Create Discord/Slack community channel
- [ ] Seed "good first issue" items for easy onboarding
- [ ] Create video tutorial for first-time contributors
- [ ] Add automated game testing (run Python/JS games in CI)
- [ ] Add algorithm/puzzle game category with examples

---

## Phase 4: Game Playground (Q3 2026)

### 🎯 Goals
- Let visitors play games directly in the browser

### 📋 Tasks
- [ ] Embed web games in sandboxed iframes
- [ ] Python CLI games via WebAssembly (Pyodide)
- [ ] "Try in browser" button on game detail pages

---

## Phase 5: Portfolio Platform (Q4 2026)

### 🎯 Goals
- Make contributions portfolio-ready for junior developers

### 📋 Tasks
- [ ] Shareable contributor profile URLs
- [ ] Contribution calendar (GitHub-style grid)
- [ ] Embeddable widget for personal portfolios
- [ ] Downloadable contribution certificate (PDF)

---

## Future Ideas (2027+)

- [ ] Partner with coding bootcamps and universities
- [ ] Mentorship matching (experienced ↔ junior)
- [ ] Game ratings and comments system
- [ ] Annual "Platform Awards" for best contributions
- [ ] AI-assisted code review for submissions
