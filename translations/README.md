# 🌍 Translations Directory

This directory contains community translations of documentation and resources.

## Structure

```
translations/
├── README/
│   ├── README.tr.md        # Turkish
│   ├── README.es.md        # Spanish
│   ├── README.fr.md        # French
│   ├── README.de.md        # German
│   ├── README.ja.md        # Japanese
│   ├── README.zh-CN.md     # Chinese (Simplified)
│   └── README.pt-BR.md     # Portuguese (Brazil)
├── CONTRIBUTING/
│   ├── CONTRIBUTING.tr.md
│   └── ...
└── guides/
    ├── BEGINNER_GUIDE.tr.md
    └── ...
```

## How to Contribute a Translation

1. Pick a document that hasn't been translated to your language yet
2. Create a new file with the correct language code suffix
3. Translate the content accurately — do NOT use machine translation only
4. Submit a PR using the **Translation** PR template
