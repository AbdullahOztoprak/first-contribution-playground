# Dice Roller

A simple CLI game that simulates rolling a six-sided dice.

How to run

Make sure you have Python 3 installed, then run:

python game.py

Example

Run the command above and press Enter to roll the dice. Type `q` and press Enter to quit.

What I Learned

- How to use `random.randint` to simulate dice rolls.
- Basic CLI input/output handling in Python.
- Writing clear, beginner-friendly code and instructions.
