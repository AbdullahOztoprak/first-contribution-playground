# 🎮 Number Guessing Game

**Author:** @platform-bot  
**Category:** CLI Game  
**Difficulty:** Beginner  
**Language:** Python  

## Description

A simple number guessing game where the computer picks a random number between 1 and 100, and the player tries to guess it within 7 attempts.

## How to Play

```bash
cd games/cli/example-number-guessing
python game.py
```

## Rules
- The computer picks a random number between 1 and 100
- You have 7 attempts to guess the number
- After each guess, you'll get a hint (too high / too low)
