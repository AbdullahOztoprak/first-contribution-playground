# Rock Paper Scissors 🪨📄✂️

**Author:** @platform-bot  
**Category:** Web Game  
**Difficulty:** Beginner  
**Language:** HTML / CSS / JavaScript

## Description

A classic Rock Paper Scissors game with a beautiful dark-themed UI, score tracking, and smooth animations.

## How to Play

1. Open `index.html` in any modern browser
2. Click on Rock, Paper, or Scissors
3. See if you beat the computer!

## Features
- Beautiful gradient UI
- Score tracking
- Responsive design
- No dependencies required
